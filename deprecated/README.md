# Deprecated skills

The skills in this folder are **deprecated and no longer maintained**. They have been superseded by the **docs-driven-dev** skill, which consolidates and extends their functionality into a single unified workflow.

**Replacement:** [docs-driven-dev](https://github.com/paulrenenichols/docs-driven-dev)

For new adopters, install and use docs-driven-dev instead.
